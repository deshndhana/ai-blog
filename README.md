# Lumina AI Blog 🧠🚀

Welcome to the **Lumina AI Blog**, a modern, premium web application built to explore the rapidly evolving world of Artificial Intelligence, Machine Learning, and Robotics.

![Lumina AI](https://picsum.photos/seed/lumina-banner/800/400)

## 🌟 Features

- **Premium Dark Mode UI**: A stunning, glassmorphism-inspired design for an optimal reading experience.
- **Dual-Language Support**: Seamlessly switch between English and Sinhala (සිංහල) for localized content.
- **Dynamic Content**: Pre-populated with over 100+ backdated AI articles spanning the last 60 days.
- **Admin Dashboard**: Secure admin panel to manage, add, and delete posts effortlessly.
- **SEO Optimized**: Fully optimized with dynamic meta tags, OpenGraph images, and an automated sitemap.
- **Responsive Design**: Looks perfect on mobile, tablet, and desktop.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS (CSS Modules & Global Styles)
- **Deployment**: [Netlify](https://www.netlify.com/)
- **Data**: Local JSON / Firebase (Configurable)

## 🚀 Getting Started (Local Development)

To run this project on your local machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/deshndhana/ai-blog.git
   cd ai-blog
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
